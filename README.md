# Goner Stock Check

## Usage 

Download my repo using `git clone`, then run `chmod +x hlg.py`. Then in a terminal run:

```python3
python3 hlg.py
```

You'll then see if there's stock or no stock. Let's see how it looks when the item you're looking for is in stock:
